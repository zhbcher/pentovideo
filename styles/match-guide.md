# 风格匹配指南

18套风格 + 26种布局，按内容类型自动推荐。写 HTML 前确定视觉身份。

## 风格速查

| 风格 | 色调 | 氛围 | 推荐场景 |
|------|------|------|---------|
| tech-dark | 深蓝+青 | 科技感 | 技术讲解/创业 |
| clean-white | 白+灰+蓝 | 清爽专业 | 企业培训/标准PPT |
| neon-gradient | 紫+粉+蓝渐变 | 年轻/潮流 | 抖音/B站/短视频 |
| business-green | 深绿+金 | 稳重商业 | 产品发布/财报 |
| warm-education | 暖橙+米白 | 亲切教学 | 教学/知识科普 |
| blue-professional | 蓝+白 | 标准商务 | 企业介绍/路演 |
| soft-editorial | 灰+柔和暖色 | 文艺/杂志感 | 品牌故事/叙事 |
| block-frame | 黑+几何色块 | 大胆/设计感 | 品牌/创意 |
| monochrome | 黑白+灰度 | 极简/高级 | 高端品牌/摄影 |
| neo-grid-bold | 荧光+深黑 | 赛博/新潮 | 科技/游戏 |
| manifesto | 大胆撞色+巨字 | 宣言感 | 品牌宣言/发布会 |
| data-dense | 深底+亮数据色 | 数据驱动 | 财报/数据报告 |
| retro-wave | 品红+青+暗紫 | 复古未来 | 音乐/创意 |
| minimal-glass | 毛玻璃+微妙色 | 精致克制 | SaaS/App展示 |
| cinematic | 暗金+电影比例 | 电影感 | 品牌大片 |
| playful-pop | 明快饱和色 | 有趣/活泼 | 儿童/消费 |
| nature-calm | 绿+大地色 | 自然/平静 | 健康/环保 |
| luxury-dark | 黑+金+微光 | 奢华 | 高端消费品 |

## 布局速查（26种）

|------|------|------|
| hero | 大标题+副标题居中 | 封面页 |

| split | 左右分屏 | 对比/双内容 |
| compare | 左右对比（产品A vs B） | 竞品对比 |
| code | 代码块展示 | 技术演示 |
| quote | 大引号+引用文本 | 名言/金句 |
| timeline | 时间线布局 | 历程/步骤 |
| steps | 步骤卡片 | 流程说明 |
| table | 数据表格 | 对比数据 |
| kpi | 大数字+标签 | 关键指标 |
| diagram | 架构图/流程图 | 技术架构 |
| pros-cons | 优劣对比 | 决策辅助 |
| image-grid | 图片网格 | 作品展示 |
| cta | 行动号召 | 结尾页 |
| thanks | 致谢 | 结尾 |
| toc | 目录 | 内容导航 |
| section | 章节标题 | 段落过渡 |
| collage | 拼贴 | 创意展示 |
| stats | 统计图表 | 数据展示 |
| panel | 面板/卡片组 | 功能罗列 |
| manifesto | 大宣言文字 | 品牌 |
| hero-illus | hero+插画 | 产品介绍 |
| bars | 柱状图/进度条 | 数据对比 |
| layout-hero | 全屏Hero | 封面 |

## 匹配规则

| 内容类型 | 推荐风格 | 推荐布局 |
|---------|---------|---------|
| 技术讲解/创业 | tech-dark, blue-professional | hero, code, diagram |
| 企业培训/标准PPT | clean-white, monochrome | steps, table, panel |
| 抖音/B站/短视频 | neon-gradient, neo-grid-bold | hero, split, stats |
| 产品发布/商业 | business-green, block-frame | hero, kpi, cta |
| 教学/知识科普 | warm-education, soft-editorial | steps, timeline, quote |
| 品牌/大胆设计 | neo-grid-bold, manifesto | hero, manifesto, collage |
