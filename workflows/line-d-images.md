# 线D：图片 → 识图 → HTML → 视频（有/无口播稿双分支）

用户提供一组图片，按顺序分配口播、配音、渲染。

## 有口播稿分支（8阶段）

```
阶段1：OCR识图 → Mistral → 逐张提取文字
阶段2：按图片顺序分配口播稿 → 均分（无需语义对齐）
阶段3：TTS配音 → narration.mp3
阶段4：语音转字幕 → transcript.json
阶段5：时间轴构建 → build_timeline.py
阶段6：写HTML → lint → validate
阶段7：预览确认 → ★ 用户确认
阶段8：渲染+交付
```

## 无口播稿分支（8阶段）

```
阶段1：OCR识图
阶段2：AI写口播稿 → voiceover.md → ★ 用户确认/修改（必须）
阶段3-8：同有口播稿分支
```

## 与线C的区别

| | 线C（PPT） | 线D（图片） |
|---|----------|----------|
| 文档转换 | ✅ LibreOffice → PNG | ❌ 已经是图片 |
| 口播对齐 | ✅ 语义匹配 | ❌ 按图片顺序均分 |
| 复杂度 | 9阶段 | 8阶段 |

## 适用场景

- 产品截图做演示视频
- 照片集做故事视频
- 设计稿做展示视频
- 漫画/绘本做配音视频
