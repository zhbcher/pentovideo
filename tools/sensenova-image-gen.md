# 商汤生图 — SenseNova Image Generation

使用 SenseNova U1 Fast 模型生成 AI 图像，用于线B（生图→视频）流程。

## 命令

```bash
python3 scripts/generate_image.py --prompt "描述" --size 2752x1536
python3 scripts/generate_image.py --prompt-file prompts/my-prompt.md --size 2752x1536
```

## 尺寸

| 尺寸 | 比例 | 场景 |
|------|------|------|
| 2048x2048 | 1:1 | 头像/图标 |
| 2752x1536 | 16:9 | PPT封面/横版视频页 |
| 1536x2752 | 9:16 | 竖版短视频 |
| 1920x1080 | 16:9 | 视频背景 |
| 1080x1920 | 9:16 | 竖版全屏 |
| 1280x720 | 16:9 | 小尺寸预览 |
| 1024x1024 | 1:1 | 社交媒体 |
| 1536x864 | 16:9 | Banner |
| 864x1536 | 9:16 | 竖版Banner |
| 2752x1152 | 2.39:1 | 宽银幕 |
| 1152x2752 | 1:2.39 | 竖版宽银幕 |

## API Key

优先级：`--api-key` → `SENSENOVA_API_KEY` 环境变量 → `openclaw.json` sensenova provider

## Prompt 模板（10类）

| 类别 | 用途 | Prompt要点 |
|------|------|-----------|
| 海报 | 产品发布/活动 | 标题区+主体+CTA |
| 信息图 | 数据展示 | 图表+数字+标注 |
| UI样机 | App/Web展示 | 设备框+界面+背景 |
| 插画 | 概念表达 | 风格+色彩+构图 |
| 产品 | 商品展示 | 角度+光影+背景 |
| 头像 | 人物/角色 | 表情+光照+风格 |
| 学术图 | 论文/报告 | 图表+标注+布局 |
| 技术图 | 架构/流程 | 框线+箭头+标签 |
| 文字排版 | 标题/封面 | 字体+层级+留白 |
| 电商 | 商品图 | 场景+氛围+卖点 |

## Prompt 编写要点

- 描述布局结构（上下/左右/居中）
- 指定配色方案
- 定义视觉风格（扁平/3D/写实/赛博朋克等）
- 列出内容清单
- 控制画面密度（不要太拥挤）
- **≥50字**
